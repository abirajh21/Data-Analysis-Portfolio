# Superstore Sales Dashboard — Excel Project

This project features an interactive Excel dashboard built using the Superstore Sales dataset.
It focuses on sales performance analysis, KPI tracking, trend exploration, and category/region comparisons.

---

## Dataset

This project uses the Superstore Sales Dataset (2015–2018) from Kaggle.

The dataset includes:

- Order details
- Customer information
- Product category and sub-category
- Sales, profit, quantity
- Region and segment details
- Ship mode and delivery times

Total records analysed: 9,800 orders

---

## Data Cleaning Steps

- Removed duplicate entries
- Standardised category, segment, and region labels
- Converted date fields to proper datetime format
- Calculated additional metrics:
  - Ship Delay (Days)
  - Average Sales per Order
- Built PivotTables for dashboard interactivity
- Created slicers for Region, Category, Segment, and Ship Mode
- Designed a clean and functional dashboard layout

---

## Key Visualisations

1. KPI Cards  
   - Total Sales  
   - Average Sales per Order  
   - Total Orders  

2. Sales by Region  
   West and East regions generate the highest revenue.

3. Sales by Category  
   Technology is the best-performing category.

4. Monthly Sales Trend  
   Shows consistent growth with seasonal fluctuations.

5. Average Ship Delay  
   Standard Class has the longest delivery delays.

6. Top 10 Products by Sales  
   Displays the highest-revenue products.

---

## Insights Summary

- Technology category leads overall revenue.
- West region consistently generates the most sales.
- Shipping delays vary significantly by ship mode.
- A small selection of products drives a large portion of revenue.
- Monthly sales show upward momentum from 2015–2018.

---

## How to Use the Dashboard

1. Open Superstore Sales Dashboard.xlsx
2. Use slicers to filter by:
   - Region
   - Category
   - Segment
   - Ship Mode
3. All charts and KPIs automatically update based on filters.
